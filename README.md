# Student Performance Prediction

## Overview
This machine learning project predicts student performance based on study habits and attendance.

## Dataset
The dataset includes the following features:
- Hours studied
- Attendance percentage
- Previous exam score

Target:
- Final exam score

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure
Student-Performance-Prediction/
├── dataset.csv
├── model.py
├── analysis.ipynb
├── requirements.txt
└── README.md

## How to Run
1. Install dependencies using `pip install -r requirements.txt`
2. Run `model.py`
3. View predictions in the output

## Outcome
The model demonstrates how student study patterns affect academic performance.
